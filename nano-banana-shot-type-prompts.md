# Shot Type Reference Images — Nano Banana Prompts

## Instructions

Generate all 30 images in one Nano Banana session to maintain subject consistency (same woman across all shots).

**Character:** Young Black woman in her early-to-mid 20s, natural hair (protective style or natural curls), casual clothing (oversized hoodie or loungewear), minimal makeup, warm skin tone. She looks like a real UGC creator, not a model.

**Setting:** Her bedroom or living room. Natural daylight from a window, slightly messy/lived-in background (fairy lights, cushions, a plant, maybe a ring light visible in some shots). Real home, not a set.

**Aesthetic:** Native TikTok/UGC feel. Phone-quality look, not cinematic. Warm tones, natural lighting, slightly imperfect framing. No TikTok UI overlays. Vertical 9:16 aspect ratio. Should look like a screenshot from a real creator's video, not a stock photo.

**Resolution:** 1080x1920 (vertical/portrait, 9:16)

---

## How to Generate

1. Open Nano Banana (via Gemini app, AI Studio, or nano-banana.ai)
2. Start with prompt 1 to establish the character
3. Generate all 30 in the same session for subject consistency
4. Use "Fast" mode for speed, or "Thinking" mode for better quality
5. Request 1080x1920 (9:16 vertical) for each
6. Download all 30 images
7. Upload to Cloudinary
8. Map each to its shot type in the lookup table

## After Upload

Create a simple lookup in n8n or a reference table in Softr:

| Shot Type | Cloudinary URL |
|---|---|
| Single (or 1-Shot) | https://res.cloudinary.com/your-cloud/... |
| Talking Head | https://res.cloudinary.com/your-cloud/... |
| Face-to-Camera | https://res.cloudinary.com/your-cloud/... |
| ... | ... |

The n8n workflow matches each frame's shot type to the URL and writes it to the Shot Example field.
